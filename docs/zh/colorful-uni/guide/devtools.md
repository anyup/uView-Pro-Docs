# 开发工具

  * 轻巧、极速
  * 强大的语法提示
  * 专为vue打造
  * 清爽护眼
  * 高效极客操作
  * markdown优先

  * 前端开发利器，轻量级，渐进式开发工具
  * 优秀的扩展工具，飞一般的代码体验

### 3. [anyup-uniui-snippet](https://marketplace.visualstudio.com/items?itemName=anyup.anyup-uniui-snippet)
