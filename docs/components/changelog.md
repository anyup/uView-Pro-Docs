## 0.0.4（2025-08-14）

### 新增

-   `u-icon` 组件新增 `space` 属性，表示`label` 在四周时与图标的距离，权重高于 `margin`，单位 rpx
-   新增`$u`工具库各类方法，同步文档
-   组件全部 setup 化，全面支持 TypeScript 和 Vue3
-   工具库示例页面全部 setup 化

### 优化

-   组件样式兼容多端
-   代码注释与类型完善
-   优化演示代码兼容性

### 修复

-   修复类型声明、变量冲突、lint 报错等问题
